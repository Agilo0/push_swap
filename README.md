# push_swap Sorting algorithm

This program sorts data in a stack using a limited set of instructions, aiming to achieve the lowest possible number of actions.

For deeper explanation, find the subject in the project.

To compile it:

make
./push_swap {the number sequence of your choice}

To compile it with the checker (on Linux):

make
./push_swap {the number sequence of your choice} | ./checker_linux {the number sequence of your choice}
